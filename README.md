# ProtonDBFinder
Util to check game for Proton compability 

## Using
``` protondb <gamename> ```
### Example
``` protondb Dota 2  ```
```id: 570
rating: Platinum
os: Ubuntu 18.04.2 LTS
notes: Actually running Proton 4.11-3. Works perfect out of the box
```
